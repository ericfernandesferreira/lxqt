Versão do LXQT : 2.3.0
Source do LXQT : https://github.com/lxqt

Requerimentos:
- slackware64 Current
- pacote de audio : https://github.com/ericfernandesferreira/audio.git

Conteúdo:
- featherpad
- kvantum
- lximage-qt
- lxqt-archiver
- lxqt-sudo
- pavucontrol-qt
- pcmanfm-qt
- qps
- qterminal
- screengrab
- icons themes (papirus,tela-circle,zafiro)
- openbox themes (arc and nordic)
- lxqt themes (less)
- kvantum themes (nordic)
- optional openbox / kwin

Esse source foi construído principalmente para quem quiser usar a última versão do LXQT com seus respectivos aplicativos para um desktop completo.

Como instalar:
- você precisa clonar com o comando : git clone https://github.com/ericfernandesferreira/lxqt.git
E então dentro da pasta criada execute o script de instalação e necessariamente precisa ser nessa ordem:

sh build-lxqt-deps.sh
sh build-lxqt-base.sh
sh build-lxqt-openbox.sh ou build-lxqt-kwin.sh
*o build lxqt-kwin ainda não está 100%, utilize o lxqt-openbox por enquanto
